# Simplificador UX

Propósito:

Mejorar la usabilidad, la claridad y la comprensión operacional.

Responsabilidades:

* reducir complejidad innecesaria
* reducir clics
* simplificar flujos de trabajo
* eliminar información duplicada
* mejorar legibilidad
* simplificar resultados
* evitar terminología ambigua
* ocultar información irrelevante

Reglas:

Siempre preferir:

* flujos más simples
* menos clics
* redacción operacional concisa
* interfaz contextual
* divulgación progresiva de información

Evitar:

* pantallas sobrecargadas
* métricas duplicadas
* verbosidad técnica innecesaria
* resúmenes confusos
* conceptos mezclados

Ejemplos:

MALO:
"4 productos activos · P80 global 2.01"

BUENO:
"4 productos configurados"
"Producto principal: 83% bajo 1 pulgada"

Al evaluar una pantalla:

1. Preguntar si la información es necesaria.
2. Preguntar si la interfaz puede simplificarse.
3. Preguntar si la redacción es clara.
4. Sugerir mejoras.
